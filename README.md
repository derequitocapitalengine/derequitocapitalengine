# Derequito Capital Engine™

## Rules-Based Investment Operating System

The Derequito Capital Engine™ is a proprietary investment operating system designed to improve the quality, consistency, and discipline of long-term investment decisions.

Instead of relying on emotion or market speculation, the engine applies a structured framework built on:

- Opportunity Ranking
- Dynamic Capital Allocation
- Position Sizing
- Portfolio Rebalancing
- Dividend Reinvestment
- Business Thesis Monitoring
- Quarterly Portfolio Reviews
- Rules-Based Governance
